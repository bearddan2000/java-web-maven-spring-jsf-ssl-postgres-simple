# java-web-maven-spring-jsf-ssl-postgres-simple

## Description
A jsf springboot java maven build,
that connects to self-signed ssl enabled postgres database.

## Tech stack
- springboot
  - web
- jsf
- maven
- bootstrap
- jquery
- dataTable

## Docker stack
- alpine
- postgres
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
